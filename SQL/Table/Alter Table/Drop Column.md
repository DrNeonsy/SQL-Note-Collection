# Dropping a Column from a Table

To drop a Column, use the `DROP COLUMN` Keyword(s).

## Command

```sql
ALTER TABLE table_name DROP COLUMN column_name;
```

## Example

```sql
ALTER TABLE Entities DROP COLUMN FavAnimal;
```
