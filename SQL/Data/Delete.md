# Deleting Data

To delete data in a Table, use the `DELETE` Keyword(s).

## Command

```sql
DELETE FROM <table_name> WHERE <condition>;
```

## Example

```sql
DELETE FROM Entities WHERE PersonID = 13;
```

## Note

For this example, you should work with [this new Entity](./Update.md).
