# Adding a Column from a Table

To add a Column, use the `ADD COLUMN` Keyword(s).

## Command

```sql
ALTER TABLE Table_name ADD COLUMN Column_name Data_type;
```

## Example

```sql
ALTER TABLE Entities ADD COLUMN BirthPlace varchar(55);
```
